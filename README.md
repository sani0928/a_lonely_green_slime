## [A lonely green slime](https://www.a-lonely-green-slime.com/)
픽셀 세계에 홀로 남은 초록 슬라임. 끝없이 몰려오는 몬스터를 피하고, 세포로 막아 내며 기어코 살아남을 것이다.. 
슬라임 조각과 뱃지를 모아 강해지고, 15분을 버티면 클리어를 달성할 수 있다!!

### 디렉토리 구조
- **root**: JS기반 웹게임 (`index.html`, `package.json`, `src/`, `assets/`, `docs/`)
- **root/backend**: Django기반 백엔드 (`scores`·`feedback`·`playlogs` API, 앱)
